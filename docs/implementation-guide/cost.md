# Custos

Todos os recursos utilizados neste projeto são elegíveis ao **AWS Free Tier** (12 meses após a criação da conta):

| Recurso | Tipo | Free Tier |
|---------|------|-----------|
| EC2 | t2.micro / t3.micro | 750 horas/mês |
| RDS PostgreSQL | db.t3.micro | 750 horas/mês + 20 GB de armazenamento |

> ⚠️ Lembre-se de **parar ou encerrar** os recursos quando não estiver usando para evitar cobranças inesperadas.

Para estimativas detalhadas, use a [Calculadora de Preços da AWS](https://calculator.aws/).
